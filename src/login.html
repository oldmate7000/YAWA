A little about this page...

This page has been created as a demo page to show use of MERN in full-stack devolopment. It's a simple weather app which takes its data from the free-tier OpenWeatherMap 5-day forecast API. Why a weather app? Because it creates a chance to demonstrate use of authentication, state management and scripting, responsive styling, database use, 3rd party API use, back-end setup and more all in one spot. If you'd like to see what went into it you may do so at the project's repository.
